DISNEY PLUS MOVIES DASHBOARD
Power BI Mini Project Documentation
________________________________________
1. Project Overview
The Disney Plus Movies Dashboard is a Power BI project created to analyze and visualize movie-related data such as published year, ratings, writers, and movie types.
The objective of this project is to transform raw movie data into meaningful insights using data cleaning, modeling, and interactive visualizations in Power BI. This dashboard helps users easily understand movie trends and performance.
________________________________________
2. Tools Used
•	Microsoft Power BI Desktop
•	Power Query Editor
•	DAX (Data Analysis Expressions)
________________________________________
3. Dataset
•	Source: Sample Movie Dataset (Disney Plus Movies Data)
•	Data Contains:
o	Movie Title
o	Published Year
o	Rating
o	Rating Type
o	Writer
o	Movie Type / Genre
________________________________________
4. Steps Followed
1.	Imported the dataset into Power BI.
2.	Cleaned the data using Power Query:
o	Removed blank values
o	Corrected data types
o	Formatted columns properly
3.	Created relationships (if multiple tables were used).
4.	Created DAX measures for calculations.
5.	Built interactive dashboard visuals:
o	Bar Charts
o	Pie Charts
o	Tables
o	Slicers
6.	Applied Top N filtering to avoid scroll bars and improve visual clarity.
7.	Designed a clean and user-friendly dashboard layout.
________________________________________
5. Key Insights
•	Identified top-rated movies in the dataset.
•	Analyzed the number of movies released each year.
•	Determined the most frequent writers.
•	Compared different rating categories.
•	Observed distribution of movies by type/genre.
________________________________________
6. Dashboard Features
